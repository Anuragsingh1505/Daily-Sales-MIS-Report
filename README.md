## MIS Sales Reporting Project (Excel)

This project is a Daily, Monthly, and Yearly MIS Sales Report designed to track business performance for a multi-channel restaurant model (Dine-in, Take Away, Delivery).

### Key Features
- Daily Sales Report (DSR)
- Monthly and Yearly Consolidation
- Target vs Actual Analysis
- KPI Tracking (Sales, Tickets, APC/APT)
- Growth Comparison with Last Year
- Automated calculations using Excel formulas

### Tools Used
- Microsoft Excel
- Advanced Formulas
- Conditional Formatting

This project demonstrates real-world MIS reporting and business performance tracking.
